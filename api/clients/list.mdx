---
title: "List Clients"
api: "GET /api/clients"
description: "Retrieve all clients for the authenticated user"
---

## Authentication

Requires authenticated user session.

## Response

Returns an array of client objects, ordered by creation date (newest first).

<ResponseField name="id" type="string">
  Unique client identifier
</ResponseField>

<ResponseField name="name" type="string">
  Client name
</ResponseField>

<ResponseField name="domain" type="string">
  Client website URL
</ResponseField>

<ResponseField name="contactName" type="string">
  Contact person name
</ResponseField>

<ResponseField name="contactEmail" type="string">
  Contact email address
</ResponseField>

<ResponseField name="googleSearchConsoleConnected" type="boolean">
  Whether Google Search Console is connected
</ResponseField>

<ResponseField name="googleAnalyticsConnected" type="boolean">
  Whether Google Analytics is connected
</ResponseField>

<ResponseField name="searchConsolePropertyUrl" type="string">
  Selected Search Console property URL
</ResponseField>

<ResponseField name="googleAnalyticsPropertyId" type="string">
  Selected Google Analytics 4 property ID
</ResponseField>

<ResponseField name="ga4PropertyId" type="string">
  GA4 property ID (alternative field)
</ResponseField>

<ResponseField name="ga4PropertyName" type="string">
  GA4 property display name
</ResponseField>

<ResponseField name="gscSiteUrl" type="string">
  Search Console site URL
</ResponseField>

<ResponseField name="gscSiteName" type="string">
  Search Console site display name
</ResponseField>

<ResponseField name="lastReportGenerated" type="string">
  ISO 8601 timestamp of last report generation
</ResponseField>

<ResponseField name="totalReportsGenerated" type="number">
  Total number of reports generated for this client
</ResponseField>

<ResponseField name="googleConnectedAt" type="string">
  ISO 8601 timestamp when Google was connected
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp when client was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp when client was last updated
</ResponseField>

<ResponseField name="reports" type="array">
  Array containing the most recent report (includes full report object)
</ResponseField>

## Example Request

```bash cURL
curl -X GET https://your-domain.com/api/clients \
  -H "Cookie: next-auth.session-token=..."
```

```javascript JavaScript
const response = await fetch('/api/clients');
const clients = await response.json();
```

## Notes

- Clients are automatically filtered to only include those owned by the authenticated user
- Results are ordered by creation date in descending order (newest first)
- Each client includes their most recent report (if any) via the `reports` array
- Empty array `[]` is returned if user has no clients
